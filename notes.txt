border-box means jitni bhi padding usko mile woh size usk height and width ke andar.

rem means font size ke root ke respect mein kitna.
agar do chiz ek hi div ke andar hai aur use mujhe ek hi line mein use krna hai toh main use karungi FLEX ko.
align-items meand cross axis(y-axis) ke center mein aa jaaye.

.boxes is display: flex

Default behavior: flex-wrap: nowrap

But when items cannot physically fit, the browser overflows and visually drops the last one

This is expected behavior

Use flex: 1 (best practice)
